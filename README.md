# projectTemplate--RedWine

在此项目中，使用 R 并运用探索性数据分析来探索红葡萄酒数据集中的一个变量或多个变量之间的关系，以及在一个选定的数据集中探索分布、异常值和反常现象。

# 项目文件
* 1、RMD 文件，它包含分析报告、最终图形和摘要，以及依次写下的反思
* 2、HTML 文件（通过 RMD 文件拼合而成）
* 3、CSV 使用的数据集
